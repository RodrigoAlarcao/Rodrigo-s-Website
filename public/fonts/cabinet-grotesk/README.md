# Cabinet Grotesk — Ficheiros de Fonte

Esta pasta precisa dos ficheiros `.woff2` do Cabinet Grotesk (Fontshare).

## Como descarregar

1. Ir a https://www.fontshare.com/fonts/cabinet-grotesk
2. Clicar em "Download Font"
3. Extrair o ZIP
4. Copiar os ficheiros `.woff2` para esta pasta com os nomes exactos abaixo

## Ficheiros necessários

| Ficheiro                              | Peso |
|---------------------------------------|------|
| CabinetGrotesk-Regular.woff2          | 400  |
| CabinetGrotesk-Medium.woff2           | 500  |
| CabinetGrotesk-Bold.woff2             | 700  |
| CabinetGrotesk-Extrabold.woff2        | 800  |

## Nota

Até os ficheiros serem adicionados, o site usa o fallback `system-ui`.
O código `next/font/local` em `app/layout.tsx` já está pré-configurado.
